# Novel-Wake-Word-Detection-Algorithm (Code is Proof-of-Concept, this is a mutli-year project not updated here)
Work in Progress. Multiple iterations ongoing. Publication pending. 
Pretrained Wav2Vec 2.0 model fine tuned on small vocabulary speech recognition system further fine tuned on a Light Gated Recurrent Unit Architecture. Stage 1: VAD on large vocabulary continuous speech recognition. Stage 2: Keyword Spotting.

For now, I've included a preview of the work to come. This project does Voice Activity Detection and Keyword Spotting using a Time-Delay-Neural-Network. All done using existing libraries (unlike most of my usual work).
